Condições meteorológicas

Selecione uma área ou use sua localização para ver o clima.
ℹ️ Os dados climáticos são fornecidos pela API Open-Meteo (gratuita, sem chave necessária) usando as coordenadas GPS da área selecionada.

Avalie se as condições climáticas atuais favorecem a aplicação de defensivos/adubo em cada talhão de cana-de-açúcar, e veja a recomendação de manejo conforme o estágio da cultura.

ℹ️ Recomendação baseada em regras gerais para cana-de-açúcar (estágio fenológico + clima atual via Open-Meteo). Não substitui análise de solo nem a avaliação do agrônomo responsável pela área. Siga sempre a bula do produto e o receituário agronômico.
Talhões de cana-de-açúcar

Configure a integração de dispositivos GPS instalados nas máquinas. Os dispositivos enviam coordenadas automaticamente para o sistema via HTTP.

Endpoint para receber posição GPS
http://127.0.0.1:8000/machines/update-location

O dispositivo GPS da máquina deve enviar um POST com o seguinte JSON a cada intervalo configurado (recomendado: 30 segundos a 5 minutos):

📋 Formato do payload JSON 
{
  "gps_id": "GPS-001",
  "lat": -18.684,
  "lng": -49.590,
  "token": "SEU_TOKEN_DE_MAQUINA",
  "velocidade_kmh": 12.4,
  "operacao": "Pulverizacao"
}
